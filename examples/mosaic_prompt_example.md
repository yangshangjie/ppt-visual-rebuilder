# Mosaic Prompt Example

```text
请基于已确认的 PPT 大纲和逐页内容，调用 $imagegen 生成 3 版不同视觉风格的 PPT 拼图方案。

任务目标：
请根据这套 PPT 的主题、行业属性、内容密度、受众场景和叙事风格，自行选择 3 种最适合的视觉方向，并分别生成一张完整拼图，方便我比较整体风格、页面节奏和高级感。

输出要求：
1. 生成 3 张 PPT 拼图。
2. 每张拼图都必须包含整套 PPT 的所有页面缩略图，并保持正确页序。
3. 每个缩略页都应是 16:9 横版 PPT 页面。
4. 三版风格必须明显不同，不要只是换颜色。
5. 每一版内部必须使用统一视觉系统，包括字体层级、色彩系统、背景风格、图表样式、图标风格、卡片/模块样式、页脚和页码样式。
6. 每一版都要像一套完整、正式、可落地的商业 PPT 方案，而不是零散页面草稿。
7. 拼图用于选择方向，文字可以适当缩小，但主标题、关键数字、核心图表和页面结构必须能看清。
8. 不要生成 PPTX，不要生成逐页单图。

推荐方向：
1. 高端科技风：黑银蓝、AI 中枢、硬件主视觉、企业驾驶舱。
2. 产品运营风：浅色背景、真实业务场景、产品 UI、实施落地感。
3. 高管汇报风：深色数据驾驶舱、KPI、战略价值、简洁有力。

请生成后停下来，让我选择方向。
```
