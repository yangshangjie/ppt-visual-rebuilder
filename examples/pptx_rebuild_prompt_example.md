# PPTX Rebuild Prompt Example

```text
所有逐页视觉稿确认。
请开始生成可编辑 PPTX。

输入：
1. 已确认的大纲。
2. 已确认的逐页高清视觉稿。
3. 用户补充的产品图、截图和品牌要求。

要求：
1. 严格参考已确认的单页视觉稿复刻。
2. 页面尺寸为 16:9。
3. 尽量使用可编辑文本、形状、图表、表格和图片，不要简单整页贴图。
4. 保持字体层级、颜色系统、背景风格、图表样式、图标风格、卡片/模块样式、页脚和页码一致。
5. 不要新增未经确认的业务数据、客户名称、金额、KPI 数字或服务承诺。
6. Dashboard / 截图类页面必须严格保留模块数量、区域比例和信息层级。
7. 生成后请渲染检查，修正错字、遮挡、重叠、溢出和页码问题。

最终文件请输出到：
outputs/pptx/final-deck.pptx
```
