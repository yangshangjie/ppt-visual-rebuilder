# Input Request Example

```text
我有一份手搓 PPT，主题是“AI 销售数字化解决方案”。

背景：
企业销售拜访过程不可视，客户沟通信息容易流失，销售经验无法沉淀。

目标：
希望把这份手搓 PPT 优化成一套正式、科技感强、适合向企业高管和销售负责人汇报的方案型 PPT。

要求：
1. 先阅读 PPT，反推大纲和逐页内容。
2. 每页包含标题、核心观点、正文要点和视觉建议。
3. 当前阶段不要生成 PPT 文件。
4. 大纲确认后，再生成 3 版不同视觉风格的整套 PPT 拼图。
5. 拼图确认后，再扩展逐页高清 16:9 单页视觉稿。
6. 关键页需要我人工确认。
```
